# 写作设定

## 风格定位

### 基调
- **整体**：轻松幽默为主，智斗热血为辅
- **第一卷（少年意气）**：以文人雅趣和市井烟火气为底色，节奏明快诙谐，参考《逍遥小书生》
- **中期（第二卷起）**：随剧情深入，逐步过渡到正剧智斗风格
- **关键冲突场景**（商战、科举舞弊、家族危机）：切换至紧张正剧风格
- **第三卷（星火燎原）**：转为热血正剧，带有军事史诗气质

### 叙事视角
- **主视角**：第三人称有限视角，主线跟随主角宋天行
- **切换时机**：关键节点切换至女主视角（柳惊鸿、苏清音等）

### 节奏偏好
- **整体策略**：快节奏爽文 + 慢热铺垫交替
- **前期（1-30章）**：快速建立矛盾与人物关系
- **中期（30章后）**：每10-15章一个小高潮，每50章一个大事件

### 文风特色
- **对话风格**：日常对话轻松诙谐，带有文人雅趣；战前/朝堂对峙时，言辞犀利、机锋暗藏
- **描写偏好**：市井烟火气与诗意写意并重；武功招式描写偏重“写意式”，如“一剑西来，天外飞仙”的意象化呈现，不过分纠缠招式招式细节
- **叙事手法**：多线并行，主线（主角视角）与支线（民间疾苦/前线战事）穿插；结尾适量留白，为续作埋钩

### 目标读者
- **读者画像**：乐于接受“末世背景下逆天改命”主题，喜爱科举升职、朝堂博弈与战场征伐的男性向网络小说读者
- **爽点密度**：中等偏上，爽点密度适中，兼顾智斗与热血情节；每10-15章一个小爽点（科举高中、商战胜利、江湖成名），每50章一个大爽点（斩杀大反派、改变战争走向）

## 核心冲突

### 主线矛盾（一句话概括）
在“偏安江南，末世将至”的时代背景下，主角宋天行从流云县秀才起步，历经科举入仕、看破腐败、杀帝起义、建立新秩序的过程，最终实现阶级跨越与民族救亡的统一。

### 分卷冲突与升级路径

#### 第一卷：少年意气（80-100章）
- **核心矛盾**：在末世背景下，通过科举和江湖手段获得生存空间与上升阶梯。主角需周旋于地方豪强、士绅势力、江湖门派之间，同时探索朝廷内部“权相”与“皇帝”的微妙关系。
- **关键转折**：察觉皇城司暗中的存在，发现秦伯渊与皇帝关系的诡异之处（看似权臣专权，实则是皇帝推出来背锅的棋子）。卷末高中状元。
- **冲突升级路径**：地方豪强压制 → 科举舞弊案 → 殿试破局 → 成名并引起朝廷注意

#### 第二卷：百姓疾苦（100-120章）
- **核心矛盾**：主角的理想（报国、为民）与腐败朝廷的矛盾激化。入朝为官后，逐步发现王朝衰落的根本原因——皇帝默许贪腐只为炼丹；前线将士浴血奋战，朝廷只求和。皇帝为炼“长生丹”取童男童女心头血作药引，触碰主角底线。
- **关键转折**：**杀皇帝**（宋天行在极度愤怒中做出抉择——杀赵煦）。皇帝一死，皇城司与秦伯渊势力火并，北戎大举南侵，天下陷入军阀割据、百姓流离的惨烈局面。主角功法秘密成为各方焦点。
- **冲突升级路径**：入朝为官发现腐败 → 襄樊前线溃败 → 皇帝取童炼药 → 触发杀帝选择

#### 第三卷：星火燎原（150章+）
- **核心矛盾**：天下崩溃，军阀割据，北戎南侵。主角需要从“江湖义军”向“人民军队”彻底转型，建立新的秩序。
- **关键转折**：以清风寨义军为骨干，深入乡村发动贫苦百姓，建立红色根据地；将旧式绿林武装改编为「星火军」；走农村包围城市、武装夺取政权的道路。
- **冲突升级路径**：北戎全面南下 → 军阀混战 → 创建根据地 → 逐鹿中原、击退北戎

## 世界观

### 世界架构

| 层级 | 内容 |
|:---|:---|
| **核心王朝** | 大炎王朝，年号「永昌」。偏安江南，仅能有效控制两淮、江南、荆湖、巴蜀、两广等地区。政令不出江南，实际影响力极为有限。 |
| **天下大势** | 大炎王朝已是落日余晖，国土仅存半壁。淮河-大散关以北全部疆域（包括旧都中都、中原腹地、关中地区）已沦陷于北戎之手。大炎与北戎形成南北对峙格局，已持续三十余年。 |
| **北戎威胁** | 北方草原部族“北戎”（架空蒙古）占据中原，定都大都。近年厉兵秣马，襄阳-樊城一线战云密布，北戎铁骑随时可能大举南侵。天下惶惶不可终日，亡国灭种之祸迫在眉睫。 |
| **时代基调** | 强烈的“末世感”。天京府（临安）歌舞升平、醉生梦死；前线将士浴血奋战，百姓流离失所。表面上繁华依旧，实则大厦将倾。 |
| **世界基调** | 类似《天龙八部》的纯粹低武江湖。无人可飞天遁地，无法术神通，无妖魔鬼怪。一切以招数、内力、实战经验为尊。 |

### 朝廷局势

| 核心人物 | 角色 | 权力脉络 |
|:---|:---|:---|
| **皇帝赵煦** | 名义上的最高统治者，实则沉迷炼丹。性情多疑自负、喜怒无常。自诩“朕虽在丹房，天下事无不知”。年轻时也曾锐意北伐，屡战屡败后心灰意冷，转而沉浸丹鼎之术，已十余年不上早朝。 |
| **权相秦伯渊** | 架空贾似道式权臣。表面代理朝政，实则皇帝推出来的“背锅侠”，专为皇帝敛财供丹鼎、背骂名。培养党羽需要皇帝的“不管事”。与皇帝关系：两头老狐狸的微妙博弈。 |
| **皇城司** | 皇帝直属密探机构（架空锦衣卫），指挥使直接听命于皇帝。通过密折制度、宦官内侍三条线，皇帝暗中掌控一切。是皇帝最信任的鹰犬，也是秦伯渊的制衡力量。 |
| **地方局面** | 朝廷控制力微弱。江南各路豪强、地方士绅、退伍将领各自结寨自保、练私兵、争地盘，但名义上仍尊朝廷号令。流云县（主角初始舞台）位于两浙西路山区，是“朝廷边缘、江湖与地方豪强势力交错”之地。 |

### 天下势力格局

| 势力类型 | 主要阵营 | 与主角关系 |
|:---|:---|:---|
| **朝廷暗流** | 大炎王朝（皇帝赵煦幕后操控、权相秦伯渊台前把持、皇城司、文官内阁、户部、刑部） | 前期：**中立偏敌**（地方官府迫害宋家）；后期：**合作与对抗并存** |
| **朝廷鹰犬** | **皇城司**（皇帝直属密探机构） | **初期敌对**（视男主为不稳定因素），后期**可转化为制衡权相的棋子**。与秦伯渊势力既合作又相互制衡。 |
| **地方豪强** | **两浙西路士绅联盟**、**流云县周家**（地方首富）、各地结寨自保的乡勇头领 | **核心敌对/博弈对象**。各势力划地为界、争抢地盘。 |
| **武林正道** | 千山域三宗：青云宗、玄剑宗、天音谷 | **合作与竞争**：女主拜入天音谷 |
| **邪道** | 血煞门、鬼影楼（暗杀组织） | **核心敌对**。意图夺取主角身上的功法或秘宝。 |
| **主角势力** | 流云县宋家（商贾）、清风寨（柳家） | **主角阵营的基础**。两家联姻共抗外敌。 |
| **隐藏势力** | 天机阁（预言组织）；老乞丐（世外高人，主角引路人） | **盟友或导师** |
| **北方强敌** | **北戎**（架空的蒙古，吞并旧北戎后建立） | **终极威胁**。当前处于对峙期，但随时可能南下，是后期主角必须面对的最大危机。 |

### 历史背景

| 时间节点 | 关键事件 | 对主线的影响 |
|:---|:---|:---|
| **百余年前（约110年前）** | 前朝大夏王朝覆灭。北戎铁骑南下攻破中都，掳走大夏皇室，史称“**中都之变**”。大炎太祖赵桓率残部于天京府（临安）重建王朝。此后大炎偏安江南，与北戎划淮河-大散关为界对峙至今。 | 大炎立国基础，半壁江山格局奠定 |
| **五十年前** | 前朝遗老密谋掀起一场**复国谋逆大案**。宋家先祖宋远游因与谋逆要员有旧，被牵连其中，遭宗族除名。他意外获得某位已故高人的武学手札，后以此为根基开创宋家基业。 | 宋家被正统世家视为“出身不正”的根源 |
| **三十年前** | 北方政权更迭。新兴的草原霸主**新北戎（对应蒙古）** 吞并旧北戎，定都大都。新北戎铁骑更凶悍好战，襄樊一线从此成为直接对抗的前线。大炎朝廷内部分裂为主战派与和议派，边防日渐废弛。 | 北戎威胁升级，战争阴影笼罩 |
| **二十年前** | 象征前朝气运的**九鼎**丢失三口，下落不明。江湖传言与北戎新主有关。 | 隐藏的世界线索 |
| **三年前** | 清风寨老寨主与宋天行父亲定下婚约（宋天行与柳惊鸿），旨在联合两家之力对抗青州军阀陈允文的欺压。 | 主角势力联盟形成 |
| **今年** | 北戎在襄樊前线调兵遣将，大战一触即发。皇帝炼丹耗费巨大，秦伯渊为讨好皇帝大肆搜刮，克扣襄樊前线军饷以供奉丹鼎。宋家与清风寨所在的两浙西路，苛捐杂税日益繁重，民不聊生。 | 故事起始时间点，末世氛围加剧 |

### 主角穿越者背景（世界观层面）
主角宋天行是穿越者，穿越前为985高校经济学硕士。这一现代知识储备对主线任务的影响：
- **第一卷**：以经济学视角看穿地方豪强盘剥模式，用于商战博弈与地方治理
- **第三卷**：以现代组织管理思想指导根据地建设，借鉴红色革命战略（农村包围城市、三湾改编式整军、土地改革等）
- **贯穿全书**：以“历史唯物主义”视角洞察王朝兴衰规律，对末世局势有超越时代的认知；但受限于低武世界社会形态，知识需“翻译”为封建语境下可理解的方案

### 地理（待补充）
- **流云县**：主角初始舞台，两浙西路边境小县。有宋府、清风寨等关键地点。
- **天音谷**：位于千山域，女主师门，以音律入武。
- **天京府**：大炎王朝行在（临安），权力中枢，歌舞升平。
- **襄樊前线**：抵御北戎的第一线，战云密布，是第二卷“百姓疾苦”的重要舞台之一。

### 规则体系
- **世界基调**：纯粹低武世界，无人可飞天遁地，无法术神通，无妖魔鬼怪。一切以招数、内力、实战经验为尊。
- **朝廷律法**：在地方豪强地盘效力有限，地方自治色彩浓厚。
- **江湖规则**：以实力和声誉为尊，门派、帮派是基层组织形式。

## 力量体系（低武设定·乔峰上限）

### 等级划分

| 境界名称 | 核心特征与实力表现 | 突破关键 |
|:---|:---|:---|
| **1.灵枢境** | 引气入体，淬炼筋骨。打通周身经脉窍穴之始，如同开启人体这座大门的“灵枢”（关键枢纽）。此境武者体魄远超常人，内力初生，可施展粗浅外功劲力。 | 感应天地之气，打通第一处关键窍穴。 |
| **2.通脉境** | 真气贯通奇经八脉，内力流转周天。此境武者轻功一跃丈余，拳脚间可附带内力震荡，已入真正武者门槛。周身经脉通畅，是为后续境界之根基。 | 疏通八脉，真气形成完整循环。 |
| **3.紫府境** | 真气凝练归元，于丹田开辟“紫府”（内息宇宙之雏形）。此境武者内力浑厚如江河，可长时间催动高深武技，甚至短暂凌空踏步、以内力隔空取物。江湖中已是赫赫有名的一方高手。 | 丹田气满，以神意开辟紫府，完成内力质变。 |
| **4.玉京境** | 紫府大成，真气化液，宛如清泉流淌于玉京（仙境，喻体内极境）。此境武者内息生生不息，护体真气凝如实质，可于体外形成罡气护罩，抵御强弓硬弩。掌力可开碑裂石，轻功可踏水而行。是当代江湖绝顶高手之列。 | 紫府臻至圆满，以心火炼化真气为真液。 |
| **5.太虚境** | **世界天花板（乔峰级别）**。返璞归真，由实入虚。内力浩瀚如渊，可沟通天地，隐隐感知万物气机流转。举手投足间皆有莫大威力，可凭内力隔空伤人于十丈之外，拳罡剑气随心而发。此境武者已是人形天灾，非军阵不可抗衡。当前时代凤毛麟角，多为隐世不出之人。 | 勘破“有无”之障，于玉京之上感悟太虚。 |

### 各卷概要

| 卷数 | 卷名 | 章节数 | 主线走向 | 核心冲突 | 关键转折 |
|:---|:---|:---|:---|:---|:---|
| 第一卷 | 少年意气 | 80-100章 | 主角在流云县从秀才起步，历经乡试→会试→殿试，最终高中状元，于文人圈中积累名望。同时与地方豪强、士绅势力博弈，以江湖游侠身份应对各方。 | 在末世背景下，通过科举和江湖手段获得生存空间与上升阶梯。探索朝廷内部微妙关系。 | 察觉皇城司暗中的存在，发现秦伯渊与皇帝关系的诡异之处。卷末高中状元。 |
| 第二卷 | 百姓疾苦 | 100-120章 | 主角入朝为官，试图凭借朝廷力量做出一番事业，但逐步发现王朝衰落与皇帝、权相的本质。襄樊前线溃败、百姓流离失所，皇帝取童男童女炼药成为最后一根稻草。 | 主角的理想（报国、为民）与腐败朝廷的矛盾激化。在“忠君”与“救民”之间做出抉择。 | **杀皇帝**。皇帝一死，皇城司与秦伯渊势力火并，北戎大举南侵，天下陷入军阀割据、百姓流离的惨烈局面。主角功法秘密暴露。 |
| 第三卷 | 星火燎原 | 150章+ | 天下崩溃，军阀割据，北戎南侵。主角以清风寨为基础，发动百姓、建立“星火军”，走农村包围城市、武装夺取政权的道路。 | 外御北戎，内平军阀，在废墟上建立新秩序。实现阶级跨越与民族救亡的统一。 | 从“江湖义军”向“人民军队”的彻底转型。 |

### 三卷路线总览

| 阶段 | 核心主题 | 主角身份 | 主要对手 |
|:---|:---|:---|:---|
