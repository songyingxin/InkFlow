## 检测任务

分析新增章节中哪些内容需要反映到设定中：

1. **世界观补充**：宏观格局 / 时代基调 / 天下大势变化（**不含具体城池**，具体地点由 locations.md 维护）
2. **力量体系变化**：新能力 / 新等级 / 新资源被揭示
3. **核心冲突推进**：主线矛盾是否有新发展
4. **卷级规划调整**：是否需要根据实际走向调整卷级概要
5. **读者承诺兑现**：开篇建立的期待是否在本章兑现了（更新兑现进度）
6. **主题母题呼应**：是否有新的母题意象出现

## 输出格式

无变化时输出：`无变化`

有变化时只输出 SEARCH/REPLACE 补丁：

```
<<<<<<< SEARCH
现有内容中的精确片段
=======
更新后的内容
>>>>>>> REPLACE
```

可输出多个补丁块。每个 SEARCH 块必须是当前设定中的**精确原文片段**。

## 原则

- **宁可漏报不可误报**——普通叙事细节不需要写入设定
- SEARCH 块必须是现有内容的精确片段，不要改写原文
- 只输出需要修改的部分，不要重新生成整个文件
- 新增的设定项可以追加到对应维度末尾（SEARCH 匹配该维度的最后一行，REPLACE 追加新行）
- 如果某个维度需要大改，说明该维度整体不适应最新走向——此时应提示用户使用 generate_settings 整体重构
