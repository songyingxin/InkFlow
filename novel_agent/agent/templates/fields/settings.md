【设定维度】
请从以下五个维度构建小说的完整设定：

## 风格定位
- 基调：轻松 / 热血 / 暗黑 / 搞笑 / 正剧 等（可多选，注明各卷/阶段的基调变化）
- 叙事视角：第一人称 / 第三人称 / 有限视角（跟随哪位角色）
- 节奏偏好：快节奏爽文 / 慢热铺垫 / 张弛有度（注明各阶段的节奏安排）
- 文风特色：对话风格、描写偏好、叙事手法等独特点
- 目标读者：读者画像与爽点密度

## 核心冲突
- 主线矛盾：推动故事的核心冲突（一句话概括）
- 分卷冲突：各卷/阶段的核心矛盾与升级路径

## 世界观
- 世界架构：空间层级结构（如大陆/位面/国家/城市等，视题材而定）
- 种族势力：各势力分布与关系
- 历史背景：关键历史事件、传说、遗迹
- 地理：重要地点、区域位置
- 规则体系：世界运行的基本法则和限制

## 力量体系
- 等级划分：完整的等级序列与晋升条件（如修炼境界、军衔、社会阶层等，视题材而定）
- 核心机制：力量获取与运用的方式（如修炼体系、科技体系、社会权力体系等）
- 特殊能力：独特的个人能力或特权
- 资源体系：关键资源与获取方式

## 卷级规划
- 全书体量：总字数/总卷数
- 各卷概要：卷名、主线走向、核心冲突、关键转折
