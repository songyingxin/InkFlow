## 七个维度

### 主题与母题（长篇灵魂）
- **主题**：这个故事本质上在讲什么（一句话）——权力 / 成长 / 救赎 / 自由 / 归属……
- **母题**：反复出现的意象或符号——雨 / 镜子 / 断剑 / 某种花……母题让长篇有文学质感
- **作者表达**：读完整个故事，读者应该带走什么思考

### 风格定位
基调、叙事视角、节奏偏好、文风特色、目标读者

### 核心冲突
主线矛盾（一句话）、各卷/阶段冲突与升级路径

### 世界观
世界架构、种族势力、历史背景、**宏观地理分区**（不含具体城池表）、基本法则

### 力量体系
等级划分、核心机制、特殊能力、资源体系

### 卷级规划
全书体量、各卷概要（卷名/主线/核心冲突/关键转折/节奏定位）

### 读者承诺（长篇连载关键）
开篇建立的核心期待，必须在结尾兑现：
- **情感承诺**：读者代入的情感方向（成长爽感 / 虐心 / 治愈 / 复仇快感）
- **悬念承诺**：开篇抛出的核心谜题（身世 / 真相 / 命运）
- **爽点承诺**：读者期待的爽点类型（打脸 / 升级 / 逆袭 / 智斗）
- **兑现进度**：当前已兑现 / 待兑现的承诺清单
