## 三个维度

### 人物关系
类型（师徒/同盟/敌对/暧昧/血亲等）、双方、现状、演变转折点、关系内伏笔、**关系中的误解/秘密**

**关系中的误解/秘密**（长篇戏剧张力来源）：
- A 以为 B 背叛了，实际 B 是被迫的 → 读者知道真相，角色不知道
- A 不知道 B 是自己的亲生父亲 → 角色不知道，读者可能知道或不知道
- 标注「读者已知」「仅 A 自知」「双方均不知」三级信息差

### 关系演变时间线
重要关系的演变轨迹——不是静态描述，而是**随章节推进的变化**：
- 格式：`初始状态 → [章N: 事件] → 当前状态`
- 示例：`挚友 → [章8: 误会] → 疏远 → [章15: 真相揭示] → 和解但留有芥蒂`
- 用于后续写作时快速定位关系发展的关键转折

### 势力关系
势力名称、主要成员、相互关系（联盟/对立/中立等）、关键事件、**势力间的秘密协议/暗流**
