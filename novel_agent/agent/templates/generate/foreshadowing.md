### 优先级（长篇核心）
- **🔴 核心**：影响主线走向 / 角色命运 / 结局的关键伏笔——必须回收，不可废弃
- **🟡 重要**：丰富剧情深度 / 强化角色 / 制造悬念——应回收，可延后
- **🟢 装饰**：氛围营造 / 细节呼应——可回收可不回收，不影响主线

### 跨卷标记
- `单卷`：本卷内埋设并回收
- `跨卷`：跨卷长线伏笔——标注预期回收卷，写作时优先追踪
- `终局`：贯穿全书、结局才回收的核心伏笔

## 五个状态区（Markdown 表格）

- 🔵 **规划中**：已记录未埋设
- 🟡 **活跃中**：已埋设待回收
- 🟢 **已回收**：填写回收章节 + 回收质量（好/弱）
- 🔴 **已废弃**：注明废弃原因
- ⚪ **已偏移**：原预期 vs 实际回收方式

## 密度控制

- 前期（1–30章）：活跃 6–10 条
- 中期（31–60章）：活跃 8–12 条
- 后期（61章+）：逐步回收，终章活跃 0–3 条
- 🔴 核心伏笔超期未回收 → 立即提醒，不可延后
- 🟡 重要伏笔超期 → 建议本卷内回收
- 超 15 条 → 合并或回收次要；不足 3 条 → 规划新伏笔

## 回收质量评估

- **好回收**：读者能记起伏笔，解释比预期更深，推动剧情
- **弱回收**：读者已忘、解释平淡、对剧情无推动
- 回收时标注质量，弱回收的 🔴 核心伏笔建议补救

## 边界（禁止写入本文件）

- 设定/角色/关系/章级大纲 → 各自文件

## 状态迁移

- 正文埋设 → 规划中 → 活跃中（填埋设章）
- 回收 → 活跃中 → 已回收（填回收章 + 质量）
- 方式偏离 → 已偏移；不再适用 → 已废弃（注明原因）
- 已回收/废弃/偏移的条目保留不删；无变化时保留原内容
