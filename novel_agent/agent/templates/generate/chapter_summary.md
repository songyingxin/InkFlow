## 摘要维度（长篇连载核心）

摘要不是"剧情简介"，而是**未来写作时的快速回忆索引**。必须能让作者在写第 50 章时，扫一眼就想起第 5 章发生了什么、谁的状态变了、埋了什么钩子。

### 两层结构

**第一层 · 事实句**（1–2 句，自然叙述，不要拆标签）

用连贯语句融合以下信息，写成一句可读的话：
- **时间**：故事内时间锚点（如「嘉靖21年春」「穿越后第3天」）。正文有明确跳跃或首次标定才写；与上章连续则省略，不写「当日清晨至午后」这类章内时段
- **地点**：主要场景或空间转移
- **人物**：出场主要角色
- **事件**：本章核心情节与不可逆变化

**第二层 · 连载索引**（接在事实句后，续写时必须对齐；无则省略）

用「标签：内容。」标注，便于扫读：
- **转折**：关键转折点。无则不写
- **状态**：角色状态变化（位置 / 心境 / 关系 / 实力 / 认知）。无变化则不写
- **伏笔**：埋设 / 回收 / 加深，用 F 编号。无则不写
- **钩子**：章末张力，下一章接笔起点

### 格式
- 用 {min_chars}–{max_chars} 字概括
- 只输出摘要正文——无标题、无章节号、无 Markdown 标题、无引号包裹
- 不编造正文没有的情节；不复制正文开头大段原文
- 第一层写自然句，第二层才用标签；不要用列表符号

### 示例

有明确时间跳跃：

```
嘉靖21年春，宋天行自清风寨独行至官道，途中遇衙役查路引，密信边角露出后用学籍脱身，官府已锁定方向。转折：侥幸→紧张。状态：宋天行对密信警觉，包袱分量压肩。伏笔：F02（密信）加深。钩子：回头见火把入林，包袱里某物滑落未察觉。
```
