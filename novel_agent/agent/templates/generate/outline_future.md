## 细纲是什么

细纲不是「剧情简介」，也不是分镜脚本，而是**续写前的写作备忘**——记录你**打算**怎么写，可随时推翻。

## 每章格式（固定 3 行）

**第 1 行 · 标题**
`### 卷名前缀 第N章 短标题`（阿拉伯数字；与已写章节标题格式一致；短标题 2–8 字）

**第 2 行 · 内容**

`- 内容：` + **一段**自然叙述（50–80 字），融合：
- **时间**：故事内时间锚点；与上章连续则省略，不写章内时段
- **地点**：主要场景或空间转移
- **人物**：本章 POV 与关键出场者
- **事件**：计划发生的 2–4 个节拍与**不可逆变化**（按时间顺序，融在句子里）

**第 3 行 · 钩子**（必写）

`- 钩子：` + 章末悬念，**具体到画面**（连载留存命脉；正文必须落到这一幕）

**第 4 行 · 伏笔**（可选，本章有伏笔动作才写）

`- 伏笔：` + 计划埋设 / 回收 / 加深，用 F 编号。本章不涉及伏笔则**整行省略**

## 示例

```
## 未来章节大纲

### 正文卷 第5章 逼债夜读
  - 内容：宋天行刚进家门，林氏还未来得及细说家事，陈允文账房便带人上门逼债。宋天行以秀才功名挡在门外，言语交锋中摸清底牌——借银滚至近四十两，对方要的是铺面。他拖延稳住局面，夜里翻父亲旧账本，发现三年前劫掠的更多疑点。
  - 钩子：天亮出门，邻家大婶塞来纸条，只写一个字——「跑」。
  - 伏笔：F03（旧账本）埋设。
```

## 注意

- 只输出 `## 未来章节大纲`，只列**未写**章节
- **禁止**分场景编号、字数分配、转折/状态标签——事件写在「内容」里
- 不写卷级总览；不写「矛盾升级」「感情推进」等空话
- 不重复 settings / characters 里已有设定
- 一次规划 **5 章**；宁可少写几章写透，不要十几章一行带过
- 与用户指定方向冲突时，体现在「内容」里

直接输出，不要代码块包裹，不要前言。
