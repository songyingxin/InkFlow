# PRD 模板

根据组件类型选择对应模板。通用章节（第 0-1 章、第 4-5 章）所有类型共用，差异章节按类型补充。

---

## 通用章节（所有类型共用）

### 第 0 章：文档信息

```markdown
# [项目名称] — 产品需求文档

## 文档信息
- 版本: v1.0
- 日期: [YYYY-MM-DD]
- 作者: [作者]
- 状态: [草稿 / 评审中 / 已确认]

## 三维度
- 项目规模: [单功能 / 大型系统]
- 组件类型: [Web产品 / AI功能 / 框架库 / 算法模型 / CLI工具 / API服务 / 数据处理]
- 实现模式: [原创 / 复现（附参考来源）]
```

### 第 1 章：需求背景

```markdown
## 需求背景

### 问题定义
[要解决什么问题？为什么现在要做？]

### 目标用户
[谁会用？使用场景？]

### 核心价值
[解决用户什么痛点？带来什么价值？]
```

### 第 4 章：验收标准

```markdown
## 验收标准

| # | 标准 | 验证方式 | 优先级 |
|---|------|---------|--------|
| 1 | [具体可验证的标准，含数值] | [自动化测试/手动操作/数值对比] | P0/P1/P2 |
```

### 第 5 章：子任务拆解

```markdown
## 子任务拆解

| # | 子任务 | 优先级 | 依赖 | 风险 |
|---|--------|--------|------|------|
| 1 | [名称] | P0 | — | [低/中/高] |
| 2 | [名称] | P0 | 1 | [低/中/高] |
```

### 第 6 章：非功能需求

```markdown
## 非功能需求

| 维度 | 要求 | 优先级 |
|------|------|--------|
| 性能 | [具体数值] | P0/P1 |
| 安全 | [具体要求] | P0 |
| 兼容性 | [具体环境] | P1 |
```

### 第 7 章：Out-of-Scope

```markdown
## Out-of-Scope（明确不做）

- [不做的事情1]
- [不做的事情2]
```
